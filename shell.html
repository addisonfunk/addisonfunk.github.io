Backup Directories

This program will take a directory as a command line argument, create a compressed archive of the directory (and create the directory if it does not exist), and name the directories in order (i.e. 1.tar.gz, 2.tar.gz, 3.tar.gz, etc.).

Backup Directories contd.

Once the program reaches 30 compressed folders, it should shift each archive back, so 1.tar.gz becomes 2.tar.gz, and so on. The remaining spot should be left for the new 1.tar.gz.
